# Code of Conduct

This project adopts a professional, respectful, and inclusive collaboration standard.

## Expected Behavior

- Be respectful and direct.
- Focus on technical clarity and evidence.
- Provide constructive feedback.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Intentional disruption of team workflows.

## Enforcement

Project maintainers may remove contributions that violate this policy.
