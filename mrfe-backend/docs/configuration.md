# Configuration Reference

## Core Runtime

- `ENVIRONMENT`: `development|staging|production|testing`
- `DEBUG`: enable debug mode
- `DATABASE_URL`: async SQLAlchemy DSN
- `MONGO_URL`, `MONGO_DATABASE`, `MONGO_NEWS_COLLECTION`
- `REDIS_URL`
## NewsAPI

- `NEWSAPI_API_KEY`
- `NEWSAPI_BASE_URL`
- `NEWSAPI_QUERY`
- `NEWSAPI_SOURCES`
- `NEWSAPI_LANGUAGE`
- `NEWSAPI_PAGE_SIZE`

## Auth/Security

- `JWT_PRIVATE_KEY`, `JWT_PUBLIC_KEY`
- `ACCESS_TOKEN_EXPIRE_MINUTES`
- `REFRESH_TOKEN_EXPIRE_DAYS`
- `SERVICE_API_KEYS_CSV`: comma-separated service keys for `X-API-Key` auth

## OpenRouter

- `LLM_PROVIDER` (default: `openrouter`)
- `OPENROUTER_API_KEY`
- `OPENROUTER_BASE_URL`
- `OPENROUTER_HTTP_REFERER`
- `OPENROUTER_X_TITLE`
- `OPENROUTER_PRIMARY_MODEL`
- `OPENROUTER_FALLBACK_MODEL`
- `OPENROUTER_TIMEOUT_SECONDS`
- `OPENROUTER_MAX_RETRIES`
- `OPENROUTER_MONTHLY_BUDGET_USD`
- `OPENROUTER_MAX_COST_USD_PER_REQUEST`

- `LOG_LEVEL`

- `RATE_LIMIT_PER_MINUTE`
